# UWA Agent Fact-Checking Framework

A modular multi-agent framework for **multimodal claim verification** (image–text), designed around:
- evidence-centric reasoning (citations for every decision),
- tool-augmented retrieval and verification,
- uncertainty-aware fusion,
- and multi-agent debate for robust final verdicts.

## Website (GitHub Pages)
This repo includes a GitHub Pages site in `/index.html`.

### Enable Pages
1. GitHub repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: `main` / `(root)` → Save
4. Your site will appear at:
   `https://<username>.github.io/<repo>/`
